# Telegram Digital Products Shop Bot

Use only for lawful digital products that you own or are authorized to distribute.

## Railway Setup

1. Upload files to GitHub.
2. Create Railway project from GitHub.
3. Add variables:
   - BOT_TOKEN
   - MONGODB_URI
   - ADMIN_IDS
4. Deploy.

## Admin First Steps

1. Start bot from admin Telegram account.
2. Open 👑 ADMIN PANEL.
3. Add payment method.
4. Add product.
5. Add stock.
6. Set support username.

## Product Delivery

Stock is one item per line. After purchase, bot instantly sends those lines to the buyer with a contact support button.
